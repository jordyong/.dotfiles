# Dotfiles

# Requirements
Ensure it is installed

Git

Stow

# Installation
Clone dotfiles repo into $HOME directory using Git
$ git clone git@github.com/jordyboy/dotfiles.git

then use GNU stow to create symlinks
$ cd dotfiles
$ stow .
